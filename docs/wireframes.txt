  B1  ████  #FFC107  (amber)
  B2  ████  #FF9800  (orange)

  C2  ████  #9C27B0  (purple)

  Again ████  #E53935 (red)
  Hard  ████  #FF8F00 (amber)
  Good  ████  #43A047 (green)
  Easy  ████  #1E88E5 (blue)

Brand:
  Primary   ████  #1A5CFF  (French blue)
  Secondary ████  #E8003D  (Tricolour red)
  Tertiary  ████  #00A36C  (success green)

Typography:
  Display / Headings : Spectral (serif) — warm, literary feel
  Body / UI text     : Inter (sans-serif) — clean, legible
  Monospace (IPA)    : Inherit system monospace

Spacing system: 4px base → 8, 12, 16, 20, 24, 32, 40, 48
Corner radii:   10px (chips), 12px (inputs), 14px (buttons), 16px (cards), 20px (sheets)
