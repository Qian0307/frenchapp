## 1. App 簡介

**法語學習 App** 是一款專為繁體中文使用者設計的法語自學工具，結合科學化的記憶方法與豐富的內容，幫助你從零開始，一路學到 C2 高級程度。

### 核心功能

| 功能 | 說明 |
|------|------|
| 🃏 **間隔重複閃卡** | 採用 SM-2 演算法，在你即將遺忘前精準複習 |
| 📖 **每日法文文章** | 依程度推送真實法文文章，單字可直接點擊查詢 |
| 📚 **文法課程** | A1 到 C2 系統化文法說明，附互動練習題 |
| 📝 **錯題本** | 自動記錄答錯的單字，集中加強複習 |
| 🔔 **智慧通知** | 每日定時提醒複習，維持學習串聯（Streak） |
| 🏆 **成就系統** | 達成學習里程碑，獲得 XP 經驗值與徽章 |

### CEFR 程度對照

```
A1 ──── A2 ──── B1 ──── B2 ──── C1 ──── C2
入門     初級     中級     中高級   高級     精通
```

本 App 收錄超過 **20,000 個法文單字**，每個單字皆提供：
- 繁體中文翻譯
- 英文翻譯（輔助參考）
- IPA 發音標注
- 真人發音音檔
- 例句（法文 + 中文）
- 用法說明與記憶口訣

## 2. 帳號註冊與登入

### 2.1 新增帳號

1. 開啟 App，點擊首頁的「**建立帳號**」
2. 填寫以下資訊：

   | 欄位 | 說明 |
   |------|------|
   | 使用者名稱 | 3–30 個字元，不可重複 |
   | 電子郵件 | 用於登入與重置密碼 |
   | 密碼 | 至少 6 個字元 |
   | 目標程度 | 選擇你想達到的 CEFR 等級（A1–C2） |

3. 點擊「**建立帳號**」完成註冊

> **小提示：** 不確定自己的程度？選 A1 開始，App 會根據你的學習進度自動調整推薦內容。

### 2.2 登入

- **電子郵件登入**：輸入帳號密碼後點擊「登入」
- **Google 登入**：點擊「以 Google 繼續」，快速授權即可
- **忘記密碼**：點擊「忘記密碼？」，系統會發送重置連結到你的信箱

### 2.3 登出

進入「**個人設定**」→ 頁面最底部 → 點擊「**登出**」

## 4. 單字閃卡系統

閃卡是本 App 最核心的功能，採用 **SM-2 間隔重複演算法**，確保你在最佳時機複習每個單字。

### 4.1 開始複習

**方法一：** 首頁點擊「繼續複習」
**方法二：** 底部導覽列點擊「🃏 卡片」→「**開始複習**」

### 4.2 閃卡操作流程

#### 第一面（法文面）
卡片正面顯示：

```
┌─────────────────────────────┐
│         [ A2 ]              │
│                             │
│      épanouissement         │  ← 法文單字（大字）
│                             │
│   /e.pa.nwis.mɑ̃/  🔊       │  ← IPA發音 + 音檔
│                             │
│         [名詞]              │  ← 詞性
│                             │
│       點擊翻面              │
└─────────────────────────────┘
```

- 點擊 **🔊** 可播放真人發音
- 嘗試在心裡說出意思後，再點擊翻面

#### 第二面（答案面）
翻面後顯示完整資訊：

```
┌─────────────────────────────┐
│    自我實現／開花結果        │  ← 繁體中文翻譯（主要）
│  fulfilment / blossoming    │  ← 英文翻譯（輔助）
│                             │
│  性別：陽性名詞              │
│                             │
│  例句：                     │
│  L'épanouissement personnel │
│  est essentiel.             │
│  → 個人的自我實現是必要的。   │
│                             │
│  💡 épanouir = 開花 → 開花  │  ← 記憶口訣
│     結果                    │
│                             │
│ [再來一次][困難][良好][簡單] │  ← 評分按鈕
└─────────────────────────────┘
```

### 4.3 評分說明

看到答案後，誠實評估自己的記憶程度：

| 按鈕 | 意思 | 下次複習 | 適用情境 |
|------|------|----------|----------|
| ✗ **再來一次** | 完全不記得 | 明天 | 完全沒印象 |
| ~ **困難** | 記得但很費力 | 稍後重新出現 | 想了很久才想起來 |
| ✓ **良好** | 正確記得 | 數天後 | 稍微想了一下就記得 |
| ★ **簡單** | 立刻記得 | 數週後 | 秒答，完全沒猶豫 |

> **重要：** 請誠實評分，評分越精確，演算法的排程就越有效率。選「簡單」不代表偷懶，而是告訴系統這張卡可以間隔更長時間。

### 4.4 複習階段說明

SM-2 演算法的運作原理：

```
第1次答對 → 1天後複習
第2次答對 → 6天後複習
第3次答對 → 15天後複習（依難易係數遞增）
...
答錯任何一次 → 重新從1天開始
```

### 4.5 加入個人牌組

在「單字瀏覽器」瀏覽時，點擊單字旁的 **⊕** 圖示，即可將該單字加入你的複習牌組。

### 4.6 星號收藏

在複習時，點擊卡片右上角的 **☆** 可以加上星號標記，之後可以從「我的牌組」→「已收藏」單獨複習。

## 6. 文法課程與練習

### 6.1 課程列表

底部導覽點擊「📚 文法」，選擇 CEFR 程度標籤，瀏覽課程清單。

每堂課顯示：
- 課程標題與簡介
- 程度標籤（A1–C2）
- 完成狀態（✓ 已完成 / 最高分）

### 6.2 課程結構

每堂課分為三個階段：

#### 第一階段：閱讀說明
- 詳細的文法概念解釋（含表格）
- 彩色標示的例句（法文 + 中文）
- 重點提示（💡 小技巧）

例：現在式 -er 動詞變化表

| 人稱 | 詞尾 | parler 範例 |
|------|------|-------------|
| je | -e | parle |
| tu | -es | parles |
| il/elle | -e | parle |
| nous | -ons | parlons |
| vous | -ez | parlez |
| ils/elles | -ent | parlent |

讀完後點擊「**開始練習**」進入第二階段。

#### 第二階段：互動練習

目前支援以下題型：

**選擇題**
```
請選出 "nous" 的正確動詞形式：
○ parle
● parlons    ← 選擇後立即顯示對錯
○ parlez
○ parlent

✓ 正確！
  nous 使用 -ons 詞尾：nous parlons。
```

**填空題**
```
填入空格：Tu ___ (habiter) à Paris.

[habites          ]  ← 輸入框
            [確認]
```

每道題答對得 **+5 XP**，答錯也會顯示正確答案及解釋。

#### 第三階段：完成畫面

```
🏆 課程完成！
分數：80%

[再做一次]  [返回課程列表]
```

- 分數 ≥ 60%：課程標記為完成，解鎖下一課
- 分數 < 60%：可重新練習，系統保留最高分紀錄
- 完成 A1 全部課程 → 自動解鎖成就「A1 基礎扎實」

### 6.3 先修課程

部分課程需要先完成前置課程才能解鎖（課程卡片上會顯示 🔒 圖示）。按照推薦順序學習，效果最好。

## 7. 錯題本

錯題本自動收集你在**閃卡複習**、**文章查詢**、**文法練習**中答錯的單字。

### 7.1 進入錯題本

底部導覽點擊「📝 錯題」

### 7.2 頁面說明

```
┌────────────────────────────────┐
│ 錯題本              [全部複習] │
├────────────────────────────────┤
│ 未解決 (12)  │  已解決 (5)     │
├────────────────────────────────┤
│ s'épanouir              ×4 🔴 │  ← 錯誤次數徽章
│ 開花；蓬勃發展                  │
│ /e.pa.nwiʁ/                   │
│                                │
│ [新增筆記]        [標記解決]   │
├────────────────────────────────┤
│ nonobstant              ×2 🔴 │
│ 儘管                           │
│ /nɔ.nɔp.stɑ̃/                  │
│ 📝 法律用語，書面正式場合才用   │  ← 個人筆記
│                                │
│ [編輯筆記]        [標記解決]   │
└────────────────────────────────┘
```

### 7.3 功能說明

#### 新增個人筆記
點擊「**新增筆記**」可以為這個單字寫下自己的記憶方法或使用情境，方便下次複習。

#### 標記解決
當你確信已經記住這個單字時，點擊「**標記解決**」，它會移到「已解決」頁籤。

> **提示：** 標記解決後，單字仍然存在你的閃卡牌組中繼續正常排程，只是從錯題本移除。

#### 全部複習
點擊右上角「**全部複習**」，進入專屬的錯題複習模式，集中攻克所有未解決的錯誤單字。

### 7.4 錯誤次數顏色說明

| 顏色 | 意思 |
|------|------|
| 🔴 紅色徽章 | 錯誤次數 ≥ 3，需要加強 |
| 🟡 橙色徽章 | 錯誤次數 2 次 |
| ⚪ 灰色徽章 | 錯誤次數 1 次 |

## 11. 常見問題（FAQ）

### Q1：閃卡為什麼今天沒有出現？
今天的複習已全部完成，或你的牌組中還沒有任何單字。請到「單字瀏覽器」加入單字，或等待明天的排程。

### Q2：我可以自己新增單字嗎？
目前版本不支援自訂單字。你可以透過「單字瀏覽器」選擇想加入的單字，或在閱讀文章時點擊單字加入牌組。

### Q3：學習串聯（Streak）斷掉了怎麼辦？
串聯歸零後會從 1 重新開始。建議開啟「串聯提醒」通知，在快要斷掉前提醒你。

### Q4：評分時選錯了（例如不小心選簡單）怎麼辦？
目前版本不支援撤銷評分，下次遇到該卡片時正確評分即可。演算法會自動修正。

### Q5：文章我看不懂怎麼辦？
- 點擊你不認識的單字查詢意思
- 開啟右上角 ▶ 聽朗讀，跟著音檔多讀幾遍
- 嘗試選擇程度更低的文章（A1 文章語法最簡單）

### Q6：離線可以使用嗎？
- 閃卡複習：部分支援（需先連線下載今日卡片）
- 文章閱讀：已緩存的最近 5 篇文章可離線閱讀
- 文法課程：需要網路連線
- 恢復網路後，離線期間的複習記錄會自動同步

### Q7：如何轉換到其他裝置？
只要使用相同帳號登入，所有學習進度（閃卡排程、文章記錄、錯題本、成就）都會自動同步，無需任何操作。

### Q8：IPA 發音標記看不懂怎麼辦？
IPA（國際音標）是一套標準的發音符號。建議：
1. 直接點擊 🔊 聽真人發音
2. 不必強記符號，多聽多模仿
3. 法語發音有一套規則，學習一段時間後自然熟悉

### Q9：我已經是進階程度，可以跳過基礎嗎？
可以。在「個人設定」→「目標程度」選擇對應等級，App 會優先推薦該程度的文章與課程。閃卡方面可在「單字瀏覽器」篩選 B2 / C1 / C2 單字加入牌組。

## 附錄：學習建議

### 每日最佳學習流程

```
起床後（5–10 分鐘）
  → 完成今日閃卡複習
  → 維持學習串聯

午休或通勤（10–15 分鐘）
  → 閱讀今日法文文章
  → 點擊生詞加入牌組

晚上（10–15 分鐘）
  → 完成一堂文法課程或練習題
  → 複習錯題本的單字
```

### 高效學習的關鍵

1. **規律 > 密集**：每天 20 分鐘比週末學 3 小時更有效
2. **誠實評分**：閃卡評分要誠實，不要為了快點結束而選「簡單」
3. **善用錯題本**：答錯不可恥，重要的是記錄下來並補強
4. **大量輸入**：法文文章不只是單字練習，要試著理解整段意思
5. **開口說**：聽完音檔後跟著念，發音從一開始就要注意
